Bedriden care robot

This project is under development. I started creating it 2 years ago. This robot is for sick people who can't get out of bed and need to take pills. It will help them at the right time take their pills by giving them a glass. Moreover you will have an opportunity to manage it by your smartphone. I created it on the Arduino platform using Arduino Mega2560 board and color sensor to recognize colors of glasses. For rotation I used stepper motor and set up the rotation angles.

Weather Bot

This Telegram-Bot was created on Python. I used technology of parsing data from the site with weather and then show them in Telegram. There you can find the weather in 6 Belarusian cities. You can get the temperature, the temperature as it felt, information about direction and speed of wind. Also to make it more beatiful I add emoji of sun, wind, etc. If something goes wrong you will get a message that "Something went wrong. Try again". You can find this project in my Github.

Game Squares

It was the final project on Informatics lessons. We have been working in C++Builder and in the end to show our skills we created the game. I made a simple game where you need to make all squares the same color. There is a stroke counter which shows how many times you change the color of squares. You can also find and download this project in my Github.
